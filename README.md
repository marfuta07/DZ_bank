# Проект "Виджет банковских операций клиента"
## Описание:
Проект " Виджет банковских операций клиента " - это виджет на Python, 
который показывает несколько последних успешных банковских операций клиента.
## Установка:
+ Установите на свой компьютер GitHub.
+ Клонируйте репозиторий с GitHub  с помощью веб-URL.
```
 git clone https://github.com/marfuta07/Homework_10_1.git
```
## Использование:
В текущем виде, созданные функции запускаются в созданных модулях. В дальнейшем весь функционал будет реализован в файле main.py


## Примеры использования функций:
+ Модуль masks

Функция маскировки номера банковской карты get_mask_card_number:
 
 **get_mask_card_number(card_number: str) -> str**

Принимает на вход номер карты и возвращает ее маску. 
Номер карты замаскирован и отображается в формате 
                   XXXX XX** **** XXXX
     Пример: 7000792289606361 → 7000 79** **** 6361

Функция маскировки номера банковского счета get_mask_account:
  
  **get_mask_account(account_number: int | str) -> str**

Принимает на вход номер счета и возвращает его маску. Номер счета замаскирован и отображается в формате 
**XXXX
     Пример: 73654108430135874305 → **4305

+ Модуль widget

Функция mask_account_card, которая умеет обрабатывать информацию как о картах, так и о счетах:

**mask_account_card(account_card: str) -> str**

Принимает  строку, содержащую тип и номер карты или счета, и возвращает строку с замаскированным номером.
     
   Пример: "Visa Platinum 8990922113665229" → "Visa Platinum  8990 92** **** 5229"

Функция get_date

**get_date(date_format: str) -> str**

Принимает на вход строку с датой в формате "2024-03-11T02:26:18.671407"
и возвращает строку с датой в формате "ДД.ММ.ГГГГ" ("11.03.2024").
      
   Пример: "2024-03-11T02:26:18.671407" → "11.03.2024"

+ Модуль processing

Функция filter_by_state, которая принимает список словарей и опционально значение для ключа state (по умолчанию 'EXECUTED'). Функция возвращает новый список словарей, содержащий только те словари, у которых ключ 
State соответствует указанному значению. 

 **filter_by_state(list_dicts: List[Dict], state: str = 'EXECUTED') -> List[Dict]:**

      
Пример: [      {'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'},
               {'id': 939719570, 'state': 'EXECUTED', 'date': '2018-06-30T02:08:58.425572'},
               {'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'},
               {'id': 615064591, 'state': 'CANCELED', 'date': '2018-10-14T08:21:33.419441'}
              ]  

→   →   [
                  {'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'},
                  {'id': 939719570, 'state': 'EXECUTED', 'date': '2018-06-30T02:08:58.425572'}
                 ]
    

Функция sort_by_date, которая принимает список словарей и необязательный параметр, задающий порядок сортировки (по умолчанию — убывание). Функция должна возвращать новый список, отсортированный по дате (date)

  **sort_by_date(list_dict: List[Dict]) -> List[Dict] :**
